# peppermint-scrot-helper

> Author: Brian Tomlinson <brian@peppermintos.com>


## Description

> A simple helper wrapper around scrot for assisting Peppermint OS users in capturing screenshots. This package is meant
> to be called via a keybinding and provides dialog-based GUI interaction via Zenity.


## Dependencies

1. scrot
2. Zenity
